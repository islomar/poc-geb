# Proof of concept of Geb for Alea

## How to execute the tests
* For Firefox: `mvn clean test -Dgeb.env=firefox`
* For Chrome: `mvn clean test -Dgeb.env=chrome`
* For HtmlUnit: WIP...

In order to execute only one test, type:
`mvn clean -Dtest=HomeSpec test`
